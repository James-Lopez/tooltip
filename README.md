# tooltip
A website with tooltips for people who want to have a better understanding of what their interactions do.

## Useage
```bash
$ npm install
$ npm run
```

## Uses
* HTML5
* CSS3
* Modern JavaScript (ES2015+)
* Express
* Node.js
